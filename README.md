# LO21-Calendar

Application C++ permettant de gérer un agenda avec la gestion de projets.

Un projet est composé de taches unitaires ou composites, qui respectent des contraintes de cohérences temporelles du projet et des autres taches.
Un interface permet de visualiser les taches programmées d'un projet.
Il est possible de créer des taches unitaires ou composites pour un projet et de les programmer.
La fonctionnalité d'export n'as pas été implémentée
